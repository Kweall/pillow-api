# Pillow Real Estate API

REST API сервис для управления недвижимостью и записи на просмотры, реализованный на Yandex Userver.

## Функциональность

- 🔐 Аутентификация пользователей (регистрация, логин, JWT токены)
- 🏠 Управление объектами недвижимости (CRUD, поиск по параметрам)
- 📅 Запись на просмотр объектов
- 👥 Управление пользователями
- 📊 In-memory хранилище данных

## Технологии

- C++20
- Yandex Userver
- OpenAPI 3.0
- Docker
